### Solution

This problem is a level order traversal problem. If you are very familiar with that problem this problem is just, (level order traversal problem + take the last element of the level i.e. `level[-1]`).

Letâ€™s remove the need for the level array to improve the space complexity.

We don't actually need the entire `level` array like we do in the prerequisite problem. We can just use need to know when we have landed on the last index of a level.

### Code

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:

        result = []

        if not root:
            return result

        q = collections.deque()
        q.append(root)

        while q:

            level = []
            _len = len(q)

            for _ in range(_len):
                cur = q.popleft()
                level.append(cur.val)
                if cur.left:
                    q.append(cur.left)
                if cur.right:
                    q.append(cur.right)

            result.append(level[-1])

        return result
```

Improved code

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:

        result = []

        if not root:
            return result

        q = collections.deque()
        q.append(root)

        while q:

            _len = len(q)

            for i in range(_len):
                cur = q.popleft()
                if i == _len-1:
                    result.append(cur.val)
                if cur.left:
                    q.append(cur.left)
                if cur.right:
                    q.append(cur.right)

        return result

```
